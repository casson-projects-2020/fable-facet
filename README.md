# fable-facet
In conjunction with Fable Facet site allows you to use your 1000 Gemini free calls to storytelling.

This is a "Open In Google Cloud Shell" repo - it will create in a Google Cloud Platform the infrastructure the site https://fablefacet.com needs to allow a user to use his own 1000 Gemini API free calls (generously given to all by Google) to generate AI-driven stories. They can be used as game - you play as the protagonist and the AI
